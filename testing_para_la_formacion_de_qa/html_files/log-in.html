Este sitio web ha sido diseñado con el objetivo de facilitar el aprendizaje de técnicas de testing en personas que estén formándose para trabajar de QA, así como permitir realizar pruebas mediante el uso de herramientas de automatización como Cypress y Selenium. Escoja un nombre de usuario de la lista que aparece a la izquierda y podrá ver la contraseña correspondiente. Introduzca los datos para acceder al sitio. Cada usuario dará acceso a una parte diferente de la página, y contendrá distintos elementos con los que interactuar.
